# ⚽ CR7 Career Goals Explorer

An interactive data app exploring every goal Cristiano Ronaldo has scored in his professional career.

## 🚀 Live Demo
[Click here](#) <!-- replace with Streamlit URL -->

## 📌 Features
- 📊 **Career Overview** — Goals by club, season, position, home vs away
- ⚽ **Goal Methods** — How and when he scores
- 🏆 **Competition Breakdown** — Champions League, leagues, cups, top opponents
- 📅 **Season by Season** — Best seasons, decade comparison
- 🔮 **Fun Stats** — Clutch goals, top assisters, free kicks, solo runs
- 🎨 **Dynamic Club Themes** — Select any club and the entire app changes color theme

## 🛠️ Tech Stack
- Python, Pandas, Plotly, Streamlit
- 816 goals across 5 clubs and 20 competitions

## 📁 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
